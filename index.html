Hosting (not Mac-only)

You do not have to run this only on your Mac. Upload this folder (HTML, JS, CSS, Master.pdf, courses.json) to any static host: GitHub Pages, Netlify, Cloudflare Pages, or your flight school website. Then open the site in Safari with an https:// URL. Google Drive “preview” is unreliable for JavaScript apps—use a real static host instead.
